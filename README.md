
## Setting up

Install dependencies

``` bash
bun install
```

Create `.env` file 

```
SHARETRIBE_INTEGRATION_SDK_CLIENT_ID=<client-id>
SHARETRIBE_INTEGRATION_SDK_CLIENT_SECRET=<client-secret>
```

```

## How to use

``` bash
bun run start <listing-id>

For example:

bun run start 67b7f55a-1dc9-4a75-8780-cb90b87e04eb
```
